# Project Docs

This directory follows a category-based convention:

- `architecture/`: System structure, design proposals, and evolution plans.
- `features/`: Feature-specific specifications and integration notes.
- `operations/`: Local runbooks, troubleshooting, and operational procedures.
- `prototypes/`: Non-production UI prototypes and experimental mockups.
